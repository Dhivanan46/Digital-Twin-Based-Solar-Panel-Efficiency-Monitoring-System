# Digital-Twin-Based-Solar-Panel-Efficiency-Monitoring-System
Digital Twin-Based Solar Panel Efficiency Monitoring System uses IoT to monitor solar panel performance in real time, detect faults like shading or damage, bypass faulty panels using MOSFET switching, send Telegram alerts, and display live efficiency data on a web dashboard.

# Components
* ESP32 Microcontroller
* INA219 Voltage & Current Sensor
* DHT11 Temperature Sensor
* IRFZ44N MOSFET
* 16x2 LCD with I2C Module
